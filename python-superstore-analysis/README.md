# 📊 Superstore Sales Analysis (Python)

## Overview
Python analysis of the Superstore dataset with visualizations
and business insights.

## Key Findings
1. Technology has the highest revenue
2. Tables, Bookcases, Supplies are losing money
3. Discounts above 20% destroy profit
4. Nov-Dec is peak season
5. West region is the top performer

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Google Colab

## Charts Created
- Bar chart (category comparison)
- Line chart (monthly trend)
- Scatter plot (discount vs profit)
- Regional dashboard (side-by-side charts)

# 📈 Stock Market Comparison Analysis

## Overview
Comparing performance of 5 top tech stocks (Apple, Google, Microsoft,
Amazon, Tesla) from 2023-2026 using statistical analysis.

## Key Findings
1. Normalized $100 investment comparison across all 5 stocks
2. Risk vs Return analysis with Sharpe Ratio
3. Correlation analysis for portfolio diversification
4. Monthly returns heatmap showing seasonal patterns
5. T-Test comparing statistical significance of returns

## Charts Created
- Price history comparison (5 stocks)
- $100 normalized investment growth
- Risk vs Return scatter plot
- Correlation heatmap
- Monthly returns heatmap
- Daily return distributions
- Statistical test results

## Skills Demonstrated
- API data collection (yfinance)
- Statistical analysis (T-Test, Sharpe Ratio, correlation)
- Multi-stock comparison & normalization
- Pandas data transformation
- Matplotlib & Seaborn visualization
- Investment insights & storytelling

## Tools
- Python, Pandas, NumPy, SciPy
- Matplotlib, Seaborn
- yfinance (Yahoo Finance API)
- Google Colab
